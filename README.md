# Movie-Time-FRT


# App features :

- "Infinite Scroll".
- Fetch Movies by categories.
- Search movies.
- Sort and filter movies.
- Default and user's lists.
- Lists are saved in localStorage
- Responsive Layout, HiRes Screens include.
